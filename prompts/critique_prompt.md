# Critique Prompt Template

## Context
You are reviewing a student-created data visualization in a university course.

## Task
Critique the chart using purpose fit, clarity, integrity, and narrative strength.

## Input
- Chart description: [insert]
- Intended audience: [insert]
- Key message: [insert]

## Output Format
1. Strengths (max 3)
2. Risks or weaknesses (max 5)
3. Top 3 prioritized revisions

## Constraints
Be specific, concise, and actionable.
