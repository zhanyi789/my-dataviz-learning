# Course File Conventions

## Naming Conventions
- Use lowercase with underscores for file names.
- Use `weekXX` format for weekly folders.
- Keep markdown files concise and sectioned with headings.

## Usage Rules
- Put reusable AI context only in `context/`.
- Put role definitions only in `agents/`.
- Put prompt templates only in `prompts/`.
- Store dataset notes and variable definitions in `datasets/codebooks/`.
- Keep project-specific work inside each `projects/*/` folder.

## Documentation Expectations
- Every project folder should have a `README.md`.
- Record major chart revisions and reasons.
- Prefer clear rationale over long narrative text.
