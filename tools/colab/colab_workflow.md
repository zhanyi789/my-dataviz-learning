# Colab Workflow Starter

## 1. Setup
- Open a new Colab notebook.
- Mount Drive if needed.
- Install any required libraries.

## 2. Load Data
- Upload or read dataset from Drive/URL.
- Check schema and missing values.

## 3. Build Visualization
- Start with a simple chart.
- Add labels, title, and annotation.

## 4. Critique and Revise
- Use critique prompt templates.
- Document improvements in markdown cells.

## 5. Export
- Save notebook and export key figures.
