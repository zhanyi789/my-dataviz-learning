# Course Context

## Setting
This is a university-level course on using generative AI in data visualization.

## Expectations
- Students apply visualization principles before stylistic choices.
- AI outputs must be reviewed, justified, and revised.
- Work should be reproducible and clearly documented.

## Typical Outputs
- Chart concepts and design rationales.
- Critique notes and revision summaries.
- Short reflections on AI-assisted decisions.
