# Prompt Writing Guide

## Recommended Structure
1. Context
2. Task
3. Input Data or Notes
4. Output Format
5. Constraints

## Quick Template
- Context: [course setting, audience, objective]
- Task: [what AI must produce]
- Input: [dataset summary, variables, draft chart]
- Output Format: [bullets, table, code, critique rubric]
- Constraints: [length, style, assumptions, do-not-do list]

## Tips
- Be explicit about audience and chart goal.
- Ask for rationale, not just output.
- Require uncertainty notes when data is incomplete.
